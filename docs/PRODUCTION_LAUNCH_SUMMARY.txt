================================================================================

STEP 1: Prepare Server (5 minutes)
  bash commands/ubuntu-setup.sh
  # OR follow SERVER_INSTALLATION_CHECKLIST.txt

STEP 2: Clone Application (2 minutes)
  git clone https://github.com/your-repo/rebookd2.git /opt/rebookd2
  cd /opt/rebookd2

STEP 3: Configure Secrets (5 minutes)
  cp .env.production.example .env.production
  nano .env.production
  # Fill in: DB_PASSWORD, ENCRYPTION_KEY, JWT_SECRET, SMTP, SMS keys

STEP 4: Deploy (15 minutes)
  bash scripts/deploy.sh
  # Automated: builds Docker, starts services, runs migrations, seeds data

STEP 5: Verify (2 minutes)
  curl http://localhost:3000/health
  # Access at http://localhost:3000
  # Login: brendanjj96@outlook.com / password1

Total Time: ~30 minutes to production!

DATABASE SETUP (ONE SQL COMMAND)
================================================================================

To set up the complete database with one command:

docker-compose -f docker-compose.prod.yml exec -T db mysql \
  -u rebookd -p$DB_PASSWORD rebookd < scripts/database-setup.sql

This creates all 23 tables with:
  • Proper indexes for performance
  • Foreign key relationships
  • Data integrity constraints
  • Default data (plans, test tenant, admin user)
  • Character set: UTF-8MB4 (emoji support)

Verify with:
  docker-compose -f docker-compose.prod.yml exec db mysql \
    -u rebookd -p rebookd -e "SELECT COUNT(*) FROM information_schema.TABLES;"
  # Should show: 23 tables
